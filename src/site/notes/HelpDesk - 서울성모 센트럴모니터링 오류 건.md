#HelpDesk #26년7월 #서울성모 #VitalSite

## 서울성모 센트럴 모니터링 오류 
1. 센트럴 모니터링 데이터 전산오류 문의 
2. 서울성모 1호기 VitalSite에 우선 접근하여 확인
3. 담당자에게 센트럴 모니터 관련 ip 문의 
	1. 10.10.40.39
	2. 10.10.40.110 
4. 10.10.40.39로 서울서울 성모 2호 VitalSite 확인 시 존재X 
5. 서울 성모 1호 VitalSite ERB ACQ가 동일 IP 확인
6. 관련 로그 내용 화면 확인시 "연결된 구성원으로 부터 응답이 없어 연결하지 못했거나, 호스트로부터 응답이 없어 연결이 끊어졌습니다." 내용 확인 
7. ![Pasted image 20260721135534.png](/img/user/Pasted%20image%2020260721135534.png)
8. 담당자에게 다시 전화하여  해당 센트럴 모니터에 대한 재시작과 네트워크 연결 상황에 대한 확인을 요청 함
9. 담당자가 이 모니터링 장비가 빈번히 문제가 발생하는것에 대한 원인을 문의 
	1. 원인에 대해서 잘 모르는상황이고 확인 사항을 전달함을 얘기 함
	2. 문제 있을 경우 다시 연락을 요청 했음
10. 서울 성모 2호기 서버  ERB/ ACQ 로그를 모니터링 중 
11. 다시 연결됨을 로그 내용으로 확인
